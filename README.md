# Task 3 - 5 Prompts What/How/Why
 Prompt 1 🎀
 WHAT: Write 3 resume bullet points for my Todo App project.
HOW: Use action verbs, add numbers, keep each point 1 line long.
WHY: I built Todo App with React and Node.js but don't know how to write it on resume.

Prompt 2 🎀:
 WHAT: Explain useState hook in React to me.
HOW: Use simple analogy + 1 small code example, explain in easy words.
WHY: I am learning React and useState is confusing for me.

Prompt 3 🎀:
 WHAT: Suggest a MERN Stack project idea for my portfolio.
HOW: Give project name, 3 main features, and tech stack used.
WHY: I need a project to show recruiters but I am a beginner.

Prompt 4 🎀:
 WHAT: Write how to answer "Tell me about yourself" in interview.
HOW: Keep answer 30 seconds, use 3-part structure: past + present + goal.
WHY: I have a junior developer interview next week and get nervous.

Prompt 5 🎀
WHAT: Write a professional LinkedIn summary for a junior web developer.
HOW: Keep it under 200 words, use a confident tone, include 3 bullet points of key skills.  
WHY: I am a beginner developer who just completed an HTML bootcamp and want to attract recruiters.
